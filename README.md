## Tutorial



### Keybindings:
#### Ctrl-S: Write
#### Ctrl-Q: Quit
#### Ctrl-U: Write Quit
#### Ctrl-F: Find
#### Ctrl-H: Help



### Keyfeatures:
#### Page up: Page up
#### Page down: Page down



### Key references:
#### Ctrl-F:
##### Press arrows to navigate through the results
##### Press <ESC> (Escape) to go back
##### Press Enter to stay at the result


### Movement is done by using the up, down, left and right arrows
